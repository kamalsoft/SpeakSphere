# SpeakSphere - Product Backlog

This document contains a prioritized list of user stories and tasks for the development team. It is intended to be a living document, updated regularly based on strategic priorities.

## Epics

- **E01**: Customer Inquiry Chatbot
- **E02**: Appointment Scheduling
- **E03**: Inventory Management
- **E04**: Multilingual Support
- **E05**: System Architecture & Infrastructure

---

## User Stories

### Priority: P0 (Must-Have for MVP)

- **(E01)** As a customer, I want to ask a question in a chat window so that I can get a quick answer to my query.
- **(E05)** As a developer, I want to set up a basic API gateway so that client requests can be routed to the correct backend service.
- **(E01)** As a developer, I want to create a chat service that can understand and respond to 5-10 basic user intents in English.
- **(E05)** As a developer, I want to provision a MongoDB database to store conversation history.
- **(E03)** As an admin, I want to be able to log in to a secure area of the application.

### Priority: P1 (High Priority)

- **(E02)** As a user, I want to ask the chatbot to schedule an appointment for me on a specific day.
- **(E02)** As a developer, I want to integrate the Appointment Service with the Google Calendar API to check for free/busy times.
- **(E04)** As a Spanish-speaking user, I want to be able to type my questions in Spanish and receive a response in Spanish.
- **(E03)** As an employee, I want to view a list of all products and their current stock levels on a dashboard.
- **(E05)** As a developer, I want to create a PostgreSQL database schema for the inventory service.

### Priority: P2 (Medium Priority)

- **(E02)** As a user, I want to receive an email reminder 24 hours before my scheduled appointment.
- **(E03)** As an inventory manager, I want to be able to update the stock count for a product through the dashboard.
- **(E01)** As a user, I want to be able to rate the chatbot's answer so the system can get smarter.
- **(E04)** As a developer, I want to add support for Hindi, Tamil, and French to the chat service.
- **(E03)** As an inventory manager, I want to receive an email alert when a product's stock level drops below 10 units.
