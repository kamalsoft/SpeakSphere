# Epic: Appointment Scheduling Service

**Phase**: 2
**Parent**: [Product Roadmap](../ROADMAP.md)

## Objective

Develop the core service for appointment scheduling, allowing users to book appointments through the chatbot. This includes integrating with the Google Calendar API to manage availability and create events.

## Key Deliverables

- **(P1)** As a user, I want to ask the chatbot to schedule an appointment for me on a specific day.
- **(P1)** As a developer, I want to integrate the Appointment Service with the Google Calendar API to check for free/busy times.
