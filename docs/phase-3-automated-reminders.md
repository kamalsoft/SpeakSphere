# Epic: Automated Appointment Reminders

**Phase**: 3
**Parent**: [Product Roadmap](../ROADMAP.md)

## Objective

Implement a system to send automated reminders for scheduled appointments via email and SMS. This will involve integrating with third-party services like SendGrid and Twilio to reduce no-shows.

## Key Deliverables

- **(P2)** As a user, I want to receive an email reminder 24 hours before my scheduled appointment.
