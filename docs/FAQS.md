# SpeakSphere - Frequently Asked Questions (FAQs)

### 1. What is SpeakSphere?

SpeakSphere is a proof-of-concept application designed to show how artificial intelligence can be used to manage customer inquiries, schedule appointments, and handle inventory. It's built to be multilingual and secure from the ground up.

### 2. Who is this project for?

It's designed for two main groups:

- **Businesses (SMBs)** looking for a modern, AI-powered tool to improve customer service and efficiency.
- **Developers** who want a reference architecture for building modular, AI-driven applications.

### 3. Is SpeakSphere open source?

Yes! The project is intended to be open source. You can find contribution guidelines in the `CONTRIBUTING.md` file.

### 4. What technologies are used?

- **Backend**: Python (with Django/Flask) for the AI and business logic services.
- **Frontend**: A modern JavaScript framework like React or Vue.js, with Chakra-UI for UI components.
- **Databases**: PostgreSQL for structured data (like inventory) and MongoDB for conversational data.
- **AI/NLP**: The project is designed to use libraries like spaCy and Hugging Face Transformers.

### 5. What languages does the chatbot support?

The goal is to support English, Tamil, Hindi, Spanish, and French. Language support is a core part of the project's design.

### 6. How do I get started with the code?

Check out the `README.md` file for detailed instructions on how to set up your development environment, install dependencies, and run the application.
