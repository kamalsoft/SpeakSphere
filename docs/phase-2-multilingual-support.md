# Epic: Multilingual Support (Spanish & Hindi)

**Phase**: 2
**Parent**: [Product Roadmap](../ROADMAP.md)

## Objective

Expand the chatbot's language capabilities by implementing support for Spanish and Hindi. This involves updating the NLP pipeline to recognize and respond in these languages.

## Key Deliverables

- **(P1)** As a Spanish-speaking user, I want to be able to type my questions in Spanish and receive a response in Spanish.
- **(P2)** As a developer, I want to add support for Hindi to the chat service.
