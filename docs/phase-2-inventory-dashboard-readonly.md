# Epic: Inventory Management Dashboard (Read-Only)

**Phase**: 2
**Parent**: [Product Roadmap](../ROADMAP.md)

## Objective

Develop the initial version of the internal inventory management dashboard. This version will provide secure, read-only access to product stock levels and will include user authentication and role-based access controls.

## Key Deliverables

- **(P0)** As an admin, I want to be able to log in to a secure area of the application.
- **(P1)** As an employee, I want to view a list of all products and their current stock levels on a dashboard.
- **(P1)** As a developer, I want to create a PostgreSQL database schema for the inventory service.
