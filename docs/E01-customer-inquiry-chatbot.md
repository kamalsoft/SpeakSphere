# Epic: Customer Inquiry Chatbot

**Phase**: 1 & 3
**Parent**: [Product Roadmap](../ROADMAP.md)

## Objective

Develop, launch, and enhance a multilingual, AI-powered chatbot capable of providing immediate, automated responses to common customer questions.

## Key Deliverables

- **(P0)** As a customer, I want to ask a question in a chat window so that I can get a quick answer to my query.
- **(P0)** As a developer, I want to create a chat service that can understand and respond to 5-10 basic user intents in English.
- **(P2)** As a user, I want to be able to rate the chatbot's answer so the system can get smarter.
